# 🎨 AI 设计排版助手 (AI Design Assistant for Photoshop)

**你的 Photoshop 全能原生外脑，让排版与调色更简单！**

在日常设计中，你是否经常为了多语种排版折行而头疼？是否总是纠结该用哪款免费商用字体？又或者在调色和构图时，希望能有一个“美术指导”帮你把把关？

**AI 设计排版助手** 是一款专为视觉设计师打造的 Photoshop 专属原生插件 (UXP)。它将强大的大语言模型无缝嵌入你的 PS 面板，帮你一站式解决**文本排版、字体选择、画面诊断与色彩重构**等日常设计难题。

---

## ✨ 核心杀手锏功能

- **✂️ 智能断句与代码块排版**
  无论是复杂的阿拉伯语、泰语、小语种，还是常规的中英文本，AI 都能根据视觉美学和语言习惯进行精准断句。断句结果会以**可一键复制的代码块**形式呈现，点一下直接粘贴进 PS 文本框，彻底告别手动敲回车！
- **🔤 免费商用字体智能推荐**
  还在到处搜“免费商用字体”？AI 会根据你的画面风格和语种，直接为你推荐最契合的免费商用字体库，规避版权风险，让商用设计更安心。
- **👁️ 一键画布诊断（附带画面发送）**
  勾选“附带发送当前画面”，AI 即可读取你当前的 PS 画布。它会化身资深美术指导，为你提供专业的**人物细节调整、整体调色/光影方向、以及排版布局优化**建议。
- **⚙️ 官方 / 第三方双擎驱动，网络无忧**
  内置可视化的接口配置面板！支持 **Google 官方原生 API** 与 **第三方代理接口（OpenAI 标准格式）** 一键切换。无论你的网络环境如何，只需填入 API Key 和模型名称即可畅快使用。
- **🖼️ 智能生图与无缝导入**
  需要灵感占位图？通过对话让 AI 生成图片后，不仅可以在面板内直接预览，点击**“发送到 PS”**，图片就会作为全新图层按比例自动插入你的当前画布中。

## 📦 安装指南

### 方法一：极简安装 (推荐)
1. 在 [Releases] 页面下载最新版本的 `.ccx` 安装包。
2. 在电脑上双击该 `.ccx` 文件，唤起 Adobe Creative Cloud 完成自动安装。
   *(注：如提示“未经验证的开发者”，请直接点击“本地安装”)*

### 方法二：开发者模式加载
如果你想自行修改代码：
1. 克隆或下载本仓库代码到本地。
2. 打开 **Adobe UXP Developer Tool**。
3. 点击右上角 `Add Plugin`，选择本仓库中的 `manifest.json` 文件。
4. 点击 `Load` 即可在 PS 中运行。

## 🚀 使用教程

### 1. 唤出面板
安装成功后，打开 Photoshop，在顶部菜单栏依次点击：
`增效工具 (Plugins)` -> `设计排版AI助手` -> `AI 设计助手`

### 2. 接口配置 (首次使用必看)
点击插件面板最顶部的 **“⚙️ 接口与模型配置”** 展开设置区：
- **如果你有原生网络条件：** 选择“官方直连 API”，填入 Google AI Studio 申请的 API Key。
- **如果你使用国内第三方代理：** 选择“第三方代理 API”，填入代理商的 URL (如 `https://api.openai.com`)、模型名称 (如 `gemini-2.5-pro` 或 `gpt-4o`) 以及代理 API Key。
- 填写完毕后，点击 **💾 保存配置并隐藏**。

### 3. 开始设计
- 在输入框中输入你的设计需求。
- 如果需要诊断当前画面，勾选 **“附带发送当前的 PS 画面进行诊断”**。
- 点击 **“发送给 AI”**，等待奇迹发生！
